## 🌊 Tentang Project

Selamat datang di project **Sistem Menu Restoran Seafood**! 🦐🐟  
Project ini dibuat untuk mensimulasikan sebuah website restoran sederhana yang menampilkan berbagai menu makanan dan minuman berbasis seafood.

Bayangkan kamu sedang membuka website restoran modern—kamu bisa melihat menu lezat seperti **Kepiting Saus Padang**, **Udang Goreng Tepung**, sampai minuman segar seperti **Es Kelapa Muda** 🥥✨

---

## 🎯 Fitur Utama

### 👀 Untuk Pengunjung (User)
- Melihat semua menu tanpa harus login
- Filter menu berdasarkan kategori:
  - 🍛 Makanan
  - 🥤 Minuman
- Tampilan modern dengan card & gambar menu
- Menu tampil acak saat memilih kategori **Semua** 🔀
- Memesan berbagai menu dengan keranjang.

---

### 🔐 Untuk Admin
- Login sistem dengan keamanan password (hash)
- CRUD Menu:
  - ➕ Tambah menu
  - ✏️ Edit menu
  - ❌ Hapus menu (dengan konfirmasi elegan)
- Upload gambar menu
- Validasi file upload (hanya gambar & ukuran dibatasi)
- Dashboard admin dengan UI modern

## 🚀 Cara Menjalankan

1. Clone repository ini
2. Pindahkan ke folder: htdocs (XAMPP)
3. Import database dari file `.sql`
4. Jalankan di browser: localhost/resto
